# miejski.pl 🐀
Kod źródłowy bota, który codziennie wrzuca filmy na kanał [@slangmiejski](https://www.tiktok.com/@slangmiejski).

## Przykładowe działanie
```sh
# "--music" - Dodaje do filmu losową muzykę z folderu "music"

python main.py --music
```
